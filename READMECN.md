# L4D2 VPK 管理器

一个为求生之路2（Left 4 Dead 2）VPK 文件设计的管理工具，基于 Flet 框架构建。

## 项目概述

本应用为求生之路2提供了一个用户友好的 VPK（Valve Pak）文件管理界面。您可以轻松浏览、提取和管理游戏资源，操作简单直观。

## 功能特性

- 浏览 VPK 文件
- 提取和管理游戏资源
- 简洁直观的用户界面

## 系统要求

- Python 3.8 或更高版本
- Flet

## 安装步骤

1. 确保已激活所需的 Conda 环境：
```bash
conda activate l4d2_vpk_manager
```

2. Flet 应该已经安装在环境中。如果未安装，请运行以下命令：
```bash
pip install flet
```

## 使用方法

运行以下命令启动应用：

```bash
python main.py
```

应用窗口将打开 VPK 管理器界面。
