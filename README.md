# L4D2 VPK Manager

A file management tool for Left 4 Dead 2 VPK files, built with Flet.

## Overview

This application provides a user-friendly interface for managing VPK (Valve Pak) files used in Left 4 Dead 2. You can easily browse, extract, and manage game resources with a simple graphical interface.

## Features

- Browse VPK files
- Extract and manage game resources
- Simple and intuitive user interface

## Requirements

- Python 3.8 or higher
- Flet

## Installation

1. Ensure you have the required Conda environment activated:
```bash
conda activate l4d2_vpk_manager
```

2. Flet should already be installed in your environment. If not, install it:
```bash
pip install flet
```

## Usage

To run the application:

```bash
python main.py
```

The application window will open with the VPK Manager interface.
